# Changelog

## [0.6.0] - 2024-04-12

- 更新: 现在使用 OpenID, 并添加 UID 野开接口支持

## [0.5.2] - 2024-03-26

- 修复: B 站弹幕链接现在需要应用心跳保活了

- 修复代码错误: err 变量在错误的作用域引用

- 添加 `/ws-info-keep`接口, 连接该 ws 接口时会持续发送心跳

## [0.4.4] - 2023-12-19

- 优化手机端登录指示文本

## [0.4.3] - 2023-11-11

- change: 改用 sse 推送事件, 因为阿里云 ws 配置没通过

## [0.4.2] - 2023-11-11

- fix: 修复 Dockerfile

## [0.4.1] - 2023-11-11

- 将大部分配置移到配置文件里了

- 移除了数据库依赖, 使用配置文件加载 OAuth Clients
